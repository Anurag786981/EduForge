# EduForge UI

Frontend application for the EduForge Coaching Institute Management Platform.

## Prerequisites

Ensure the following software is installed before running the application:

* Node.js 22+
* npm 10+
* Angular CLI 19+

Verify installation:

```bash
node -v
npm -v
ng version
```

## Getting Started

### 1. Clone the Repository

```bash
git clone <repository-url>
cd frontend/eduforge-ui
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start the Application

```bash
ng serve
```

The application will be available at:

```text
http://localhost:4200
```

### 4. Build for Production

```bash
ng build
```

Build artifacts will be generated inside the `dist/` directory.
